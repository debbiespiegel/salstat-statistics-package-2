Salstat2

Descripción

Salstat2 es un software escrito en python y diseñado para el usuario final posee una interfase gráfica de usuario y se puede programar, es multiplataforma, tiene un sistema de gráficos heredado de Matplotlib, permite el uso de diferentes librerías como numpy para los cálculos numéricos,  también permite interactuar con Microsoft Excel usando un cliente com en la plataforma Windows (R) también es posible crear sus propios cuadros de diálogo utilizando el panel de script o el shell interactivo.

Noticias

  • 04-Jun-2013: versión inicial

Funciones estadísticas disponibles

Las siguientes son algunas de las funciones estadísticas disponibles:

  • Variabilidad: samplevar, sample stddev, var, stddev, signaltonoise,
  • Anova one way
  • Cálculos de probabilidad: chisqprob, ercfunc, zprob func, betacf, betai
  • Funciones de correlación: pearsonr, spearman, pointbiserialr, kendalTau, linregress
  • Momentos: variation, skew, kurtosis, skew test, kurtosis test, normal test
  • Estadística indiferencial
  • Estadísticas de frecuencia
  • Medidas de tendencia central: mean, geometric mean, harmonic mean, median mode
  • Rrdenar los datos
  • Six Sigma, para ser usado en análisis global de datos continuos.

Sistema de gráficos:

Mediante el sistema de gráficos usted tiene a disposición alrededor de 20 diferentes tipos de gráficos, al mismo tiempo usted puede interactuar con algunas características de ellos como lo son el título, dar nombre al las X, al eje en la Y, encender o apagar la malla, cambiar los colores de las líneas y mucho más. Algunos de los tipos de gráficos son:


Box & whisker, Linear regresion plot, Normal probability plot, Scatter chart, Ternary scatter chart, Histogram chart - optionally you can draw a normal curve, Multiple are chart, Bar chart of all means, Line chart of all means, bar chart, Horizontal bar chart, stacked bars, Cumulative histogram chart, Stem chart, area chart, control chart, Line charts of the data, adaptive plot, pareto Chart since S2 v2.1 rc2.

Es posible guardar los gráficos en los siguientes formatos:

eps, emf, jpeg, jpg, pdf, png, ps, raw, rgba, svg, svgz, tif, tiff

Sistema de salida mejorado:

todos los resultados numéricos se envían en una hoja del panel de salida donde es posible copiar, pegar, cortar y editarlos. Si el texto en la celda es muy largo y no lo puede ver entonces puede utilizar una barra de herramientas diseñado para ello.

Tabla pivote

El programa provee una clase para manejar los datos como una tabla pivote. Actualmente se encuentra en desarrollo la interfase gráfica de usuario para administrarla.

Shell interactivo:

el programa provee un Shell interactivo desde donde se puede manipular varios de los objetos del programa como son:

  • El panel de ingreso.
  • El panel de respuesta.
  • Las estadísticas.
  • La librería numpy.
  • Crear cuadros de diálogo.

Panel de transformación:

SEI campamento proporciona un panel de transformación en el cual se pueden realizar diferentes cálculos con las columnas del panel principal además permite utilizar diferentes funciones estadísticas incorporadas.



Creación de cuadros de diálogo:

es muy fácil crear cuadros de diálogos utilizando una clase apropiada llamada dialog.por ejemplo, ejecutando el siguiente escrito desde el Shell:

txt1= ['StaticText',['input the average']]
btn1= ['NumTextCtrl', []]
structure= list()
structure.append([txt1, btn1])
dlg= dialog(struct= structure)
dlg.Show()
