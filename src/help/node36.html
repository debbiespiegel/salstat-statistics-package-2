Spearmans Rank Correlation

Spearmans rho is a correlation for non parametric data which unlike Pearsons correlation uses the rank ordering of the data, and is therefore not susceptible to outliers in the data.

Data are entered in one column for each sample. Go to the menu, click on ``Analyse'' and then ``2 Condition Tests...'' and a new window will appear. Using the 2 drop down boxes at the top of the window, select which columns contain the data, check ``Spearmans Correlation'' from the box marked ``Non Parametric Tests'', select a one or two tailed hypothesis (two tailed is selected by default), and click on the ``Okay'' button.

The output consists of the degrees of freedom (in brackets), a value for rho (which shows the correlation direction and strength), and a p value.
